## Overview

Entities represent all moving objects in Minestom including players, mobs, items, and projectiles.

---

## Entity Class

Base class for all entities in the game.

### Entity Creation

<ParamField path="Entity(EntityType)" type="constructor">
  Creates a new entity with a random UUID.
  
  **Parameters:**
  - `entityType` - The type of entity to create
  
  ```java
  Entity zombie = new EntityCreature(EntityType.ZOMBIE);
  ```
</ParamField>

<ParamField path="Entity(EntityType, UUID)" type="constructor">
  Creates an entity with a specific UUID.
  
  **Parameters:**
  - `entityType` - The entity type
  - `uuid` - The entity UUID
</ParamField>

### Spawning & Despawning

<ParamField path="setInstance(Instance, Pos)" type="CompletableFuture<Void>">
  Spawns the entity in an instance at the specified position.
  
  **Parameters:**
  - `instance` - The instance to spawn in
  - `spawnPosition` - The spawn position
  
  **Returns:** Future completed when entity is spawned
  
  **Throws:** IllegalStateException if instance is not registered
  
  ```java
  Entity entity = new EntityCreature(EntityType.ZOMBIE);
  entity.setInstance(instance, new Pos(0, 42, 0)).thenRun(() -> {
      System.out.println("Entity spawned!");
  });
  ```
</ParamField>

<ParamField path="setInstance(Instance)" type="CompletableFuture<Void>">
  Changes the entity's instance, keeping current position.
  
  **Parameters:**
  - `instance` - The new instance
  
  **Returns:** Future completed when transfer is done
</ParamField>

<ParamField path="remove()" type="void">
  Removes the entity temporarily (can respawn).
  
  ```java
  entity.remove();
  ```
</ParamField>

<ParamField path="remove(boolean)" type="void">
  Removes the entity with optional permanent removal.
  
  **Parameters:**
  - `permanent` - true for permanent removal
</ParamField>

### Position & Movement

<ParamField path="teleport(Pos)" type="CompletableFuture<Void>">
  Teleports the entity to a position.
  
  **Parameters:**
  - `position` - The target position
  
  **Returns:** Future completed when teleport finishes
  
  ```java
  entity.teleport(new Pos(10, 64, 10));
  ```
</ParamField>

<ParamField path="teleport(Pos, Vec)" type="CompletableFuture<Void>">
  Teleports with velocity.
  
  **Parameters:**
  - `position` - Target position
  - `velocity` - New velocity vector
  
  **Returns:** Future completed when teleport finishes
</ParamField>

<ParamField path="getPosition()" type="Pos">
  Gets the current entity position.
  
  **Returns:** The entity position (includes yaw/pitch)
  
  ```java
  Pos pos = entity.getPosition();
  System.out.println("Entity at: " + pos.x() + ", " + pos.y() + ", " + pos.z());
  ```
</ParamField>

<ParamField path="setVelocity(Vec)" type="void">
  Sets the entity velocity and fires EntityVelocityEvent.
  
  **Parameters:**
  - `velocity` - The velocity in blocks/second
  
  ```java
  entity.setVelocity(new Vec(0, 10, 0)); // Launch upward
  ```
</ParamField>

<ParamField path="getVelocity()" type="Vec">
  Gets the entity velocity.
  
  **Returns:** Velocity vector in blocks/second
</ParamField>

<ParamField path="setView(float, float)" type="void">
  Changes the entity's view direction.
  
  **Parameters:**
  - `yaw` - Horizontal rotation
  - `pitch` - Vertical rotation
  
  ```java
  entity.setView(90f, 0f); // Face east
  ```
</ParamField>

<ParamField path="lookAt(Point)" type="void">
  Makes the entity look at a specific position.
  
  **Parameters:**
  - `point` - The point to look at
  
  ```java
  entity.lookAt(player.getPosition());
  ```
</ParamField>

<ParamField path="lookAt(Entity)" type="void">
  Makes the entity look at another entity.
  
  **Parameters:**
  - `entity` - The entity to look at
  
  **Throws:** IllegalArgumentException if entities are in different instances
</ParamField>

  - `invisible` - true to make invisible

<ParamField path="setGlowing(boolean)" type="void">

## EntityType

Represents a type of entity (autogenerated from Minecraft data).

<ParamField path="fromKey(String)" type="static EntityType">
  Gets an EntityType by namespaced key.
  
  **Parameters:**
  - `key` - The entity key (e.g., "minecraft:zombie")
  
  **Returns:** The EntityType, or null
  
  ```java
  EntityType zombie = EntityType.fromKey("minecraft:zombie");
  ```
</ParamField>

<ParamField path="values()" type="static Collection<EntityType>">
  Gets all registered entity types.
  
  **Returns:** Collection of all EntityTypes
</ParamField>

## Examples

<CodeGroup>
```java Spawning Entities
public class EntitySpawning {
    public static void spawnZombie(Instance instance, Pos pos) {
        EntityCreature zombie = new EntityCreature(EntityType.ZOMBIE);
        zombie.setInstance(instance, pos);
        
        // Equip the zombie
        zombie.setHelmet(ItemStack.of(Material.DIAMOND_HELMET));
        zombie.setItemInMainHand(ItemStack.of(Material.DIAMOND_SWORD));
        
        // Make it glow
        zombie.setGlowing(true);
    }
}
```

```java Player Management
public class PlayerManager {
    public static void setupPlayer(Player player, Instance world) {
        // Set spawn instance and position
        player.setRespawnPoint(new Pos(0, 64, 0));
        
        // Configure abilities
        player.setAllowFlying(true);
        player.setGameMode(GameMode.CREATIVE);
        
        // Give items
        player.getInventory().addItemStack(
            ItemStack.of(Material.DIAMOND_SWORD)
        );
        
        // Send welcome message
        player.sendMessage(
            Component.text("Welcome to the server!")
                .color(NamedTextColor.GOLD)
        );
    }
}
```

```java Entity AI
public class SimpleAI {
    public static void trackPlayer(LivingEntity entity, Player target) {
        entity.scheduleNextTick(self -> {
            if (self.getInstance() == target.getInstance()) {
                // Look at player
                self.lookAt(target);
                
                // Move towards player
                Vec direction = target.getPosition()
                    .sub(self.getPosition())
                    .normalize();
                self.setVelocity(direction.mul(3)); // 3 blocks/sec
            }
            
            // Schedule next tick
            trackPlayer((LivingEntity) self, target);
        });
    }
}
```
</CodeGroup>
